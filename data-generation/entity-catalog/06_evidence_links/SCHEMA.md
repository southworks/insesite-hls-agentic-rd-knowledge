# 06 Evidence Links Schema

Curated links between normalized entities, each backed by Raw Layer evidence.

## Required fields

- `link_id`
- `document_type` = `evidence_link`
- `source_entity_id`
- `target_entity_id`
- `relation_type`
- `rationale`
- `evidence_strength`
- `raw_sources`
- `provenance`
