# Agent Input Evidence Cards

Optional multi-format replicas for extraction-consistency testing. Generated by `generate_agent_documents.py` into:

```
corpus/txt/agent_inputs/<category>/<document_id>.txt
corpus/md/agent_inputs/...
corpus/html/agent_inputs/...
corpus/pdf/agent_inputs/...
```

Manifest: `corpus/agent_document_manifest.json`

## Source of truth

Cards are derived from normalized entities built in memory (`build_entity_index()` in `generate_normalized_layers.py`). They preserve the same facts as the JSON entities and trace back to `corpus/` raw files.

## When to run

```bash
cd data-generation/scripts
python3 generate_agent_documents.py
python3 build_case_folders.py   # ING-004 uses txt under agent_inputs/curation_decisions/
```

Demo ingest uses flat upload files in `rd-knowledge-mining/backend/dataset-seed/cases/*/ingest/` — not the full agent_inputs tree.
