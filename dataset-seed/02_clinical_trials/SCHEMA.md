# 02 Clinical Trials Schema

Clinical trial entities derived from ClinicalTrials.gov study JSON and protocol/SAP PDFs.

## Required fields

- `trial_id`
- `document_type` = `clinical_trial`
- `source_system`
- `acronym`
- `brief_title`
- `overall_status`
- `lead_sponsor`
- `phases`
- `enrollment`
- `conditions`
- `arms`
- `interventions`
- `primary_outcomes`
- `has_results`
- `public_documents`
- `raw_sources`
- `privacy_posture`
