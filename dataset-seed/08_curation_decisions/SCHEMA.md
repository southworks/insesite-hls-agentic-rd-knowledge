# 08 Curation Decisions Schema

Allow/deny/review decisions for source inclusion and compliance guardrails.

## Required fields

- `decision_id`
- `document_type` = `curation_decision`
- `source_id`
- `source_type`
- `decision`
- `reason`
- `required_human_review`
- `policy_refs`
- `curator`
- `decision_date`
- `raw_sources`
- `provenance`
