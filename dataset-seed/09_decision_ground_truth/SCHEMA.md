# 09 Decision Ground Truth Schema

Expected agent outcomes for evaluating ingestion, linking, retrieval, and curation flows.

## Required fields

- `scenario_id`
- `document_type` = `decision_ground_truth`
- `expected_agent`
- `expected_decision`
- `primary_reason`
- `top_policy_refs`
- `source_entities`
- `expected_outputs`
- `required_human_review`
- `risk_flags`
- `summary_explanation`
- `raw_sources`
